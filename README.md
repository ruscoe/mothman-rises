# Mothman Rises

A game about Mothman.

## Requirements

* nodejs
* npm
* typescript (npm install -g typescript)

## Building

Clone this repo.

`git clone git@github.com:ruscoe/mothman-rises.git`

`cd mothman-rises`

Compile the JavaScript.

`tsc`

Open `index.html` in your brower of choice.
